# Car Price Prediction

## Overview
This project predicts car prices using machine learning techniques based on various features such as engine size, horsepower, and body style.

## Objectives
- Analyze factors affecting car prices
- Perform data cleaning and EDA
- Build and compare ML models
- Optimize model using hyperparameter tuning

## Models Used
- Linear Regression
- Decision Tree
- Random Forest
- Tuned Random Forest (Best Model)

## Evaluation Metrics
- MAE
- RMSE
- R² Score

## Results
Tuned Random Forest performed the best with highest accuracy and lowest error.

## Key Insights
- Engine size, horsepower, and curb weight strongly influence price
- Feature engineering improved model performance

## Risks & Improvements
- Encoding bias may affect results
- One-Hot Encoding can improve performance

## Conclusion
The model helps in understanding pricing patterns and supports data-driven decision making.
